# Heroes UI

## Версия Unity
**2022.3.62f2** (LTS)

## О проекте
UI-прототип с анимированным экраном выбора навыков.

## Скриншот
![Скриншот экрана из Unity](Screenshots/unity.png)

## Возможности
- Селектор навыков с плавным перемещением индикатора
- Анимации появления экрана (выезд, затухание)
- Кастомный UI-шейдер градиентного эллипса

## Технологии
- **Рендер:** URP 2D
- **Анимации:** DOTween
- **Текст:** TextMesh Pro
